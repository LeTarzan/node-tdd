# Simple To-Do List using TDD

Developed To-Do list using using TDD (test-driven development) and concepts like Mock, Stub and dependency injection.

# Run tests

In root run:

```console
foo@bar:~$ npm install
```

* General test: 

```console
foo@bar:~$ npm test
```

* Development testing:

```console
foo@bar:~$ npm run test:dev
```

* Check test coverage

```console
foo@bar:~$ npm run test:cov
```
